## 📋 Getting Started

### Prerequisites
- Clarinet CLI installed
- Node.js and npm
- Git

### Installation
```bash
git clone https://github.com/mac/entropy-reversal-energy-protocol.git
cd entropy-reversal-energy-protocol
npm install
```

### Testing
```bash
clarinet test
npm test
```

### Deployment
```bash
clarinet check
clarinet deploy --testnet
```
